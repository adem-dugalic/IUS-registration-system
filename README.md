# IUS registration system
 Full registration system for International University of Sarajevo built using Next.js


# Preffered manager yarn

To start first run 

```
yarn
```
or

```
npm install
```
to install all the packages.


After the installation run 
```
yarn dev
```
or 
```
npm dev
```
